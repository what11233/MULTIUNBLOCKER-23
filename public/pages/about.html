Q: Will this work on my Chromebook?
A: Yes! If you're using GitHub Codespaces, it works perfectly. You can also run it locally on any Linux/Mac/Windows computer.

Q: Can I use this offline?
A: The unblocker needs internet to fetch websites. Other tools like text conversion, code editor, and generators work offline.

Q: What sites work with the unblocker?
A: Most websites work great! Some sites with advanced security (YouTube, Facebook, Discord) may not work due to their security protocols.

Q: Is my data safe?
A: All data stays on your device. The proxy server doesn't log or store any information.

Q: Can I deploy this online?
A: Yes! You can deploy to Render, Heroku, or any Node.js hosting service.
